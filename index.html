The Enchanted Drawing
was the first animated motion picture and live action film created by J. Stuart Blackton. This marked the start of the History of Animation and led Blackton to be considered the father of American animation.
1915
Cel Technique Invented
by John Bray. Through this, he was able to create moving animations on transparent cel sheets. This allowed for animations to be created much easier and speed up the animation process.
